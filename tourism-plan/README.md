# Plan with Love

A simple React application for browsing and managing Indian tour packages.

## Features

- Browse a list of popular Indian destinations
- View detailed information and pricing for each tour
- Remove tours from the list
- Refresh to restore all tours

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Run `npm install` to install dependencies
4. Run `npm start` to start the development server

## Technologies Used

- React
- JavaScript
- CSS

## Usage

Open the app in your browser and explore the available tours. Click the "Not Interested" button to remove a tour, or use the "Refresh" button to restore all tours.
